Asides

Sometimes in the normal flow of an article you might have something that you want to say thats related but not part of the main topic. In these cases a margin note can be particularly useful. They display information to the side of the content so that inspecting it requires just a shift of the eyes. This is in contrast to footnotes which work much worse on web browsers. The reason being that a web browser is generally one continuous page so you would need to scroll all the way down to read a footnote.
